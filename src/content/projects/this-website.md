The first project on this site is the site itself.

**Goal:** a fast, minimal personal site where adding content means
writing a Markdown file — nothing more.

**Stack:**

- [Astro](https://astro.build) compiles Markdown into plain HTML pages
- One hand-written CSS file, no framework
- Hosted free on GitHub Pages, deployed automatically on `git push`
